About this website

This website is mostly hand-written in HTML (because Markdown lacks some typographic features such as the non-breaking space and citations tags).

I do my best to make this website fast, lightweight, accessible, and privacy-friendly.

  • Lightweight: no external font, small CSS (no framework)
  • No tracker in use, and no external script.
  • Distributed over a CDN supporting efficient protocols.
  • High color contrast both in light and dark mode (ranked AAA by Firefox's Web Developer Tools.)

Tech & Open Source

The source code of this website is open source and distributed under CeCILL-B license (permissive BSD-like).

This website is generated using a Python script that produces a build.ninja file which is then processed by the Ninja build system.
