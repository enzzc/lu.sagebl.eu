About this website

This website is mostly hand-written in HTML (because Markdown lacks some typographic features such as the non-breaking space, citation tags, etc.)

I do my best to keep this website fast, lightweight, accessible, and privacy-friendly.

  • Lightweight: no external font, small CSS (no framework).
  • No tracker in use, and no external script.
  • Distributed over a CDN supporting efficient protocols.
  • High color contrast both in light and dark mode.
  • Use of semantic tags and attributes. (Well, when I am aware of them, to be honest.)

Tech

The source code of this website is open source and distributed under CeCILL-B license (permissive BSD-like).

This static website is generated with a Python script that produces a build.ninja file which is then processed by the Ninja build system to generate the final HTML files.
