Software projects

Right now I'm working actively on these projects. Most of them are software projects written in Go or Python.

  • Hassaku: API/SaaS providing automagic audio enhancement. Currently a work in progress, but it should be ready soon!
  • ZLS: protocol and tools to do outdoors livestreaming reliably and with QoS
  • Rivulet: Web-based minimalist RSS/Atom/JSON Feed aggregator (my public instance)
  • No-nonsense and privacy-friendly Web speedometer (source code). It has been tested on board a TGV (high-speed train) with Android successfully.
  • idhider: experimental ID obfuscating scheme based on the Blowfish cipher. It is an alternative to using UUIDv4 as primary keys (such in a database) to avoid disclosing IDs publicly.
  • Deluge: high-troughput message streaming platform supporting HTTP and gRPC
  • hurl: hashed URL lookup local database. It acts like a simple key-value store. The layout is inspired by the Git object database.
